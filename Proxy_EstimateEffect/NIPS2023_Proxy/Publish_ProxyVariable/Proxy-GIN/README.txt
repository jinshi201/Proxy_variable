Proxy-GIN-Main.py
Main Funcion: Proxy_GIN(X, Y, q, data, alpha)

# X: treatments, datatype: list; e.g., X=['x1','x2']
# Y: outcome, data type: string; e.g., Y='y'

#data: dataset, datatype: DataFrame
#alpha: confidence threshold
